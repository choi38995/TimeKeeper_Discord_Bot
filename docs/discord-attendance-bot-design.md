# Discord Attendance Bot 설계 문서

## 1. 프로젝트 개요

이 프로젝트는 Discord 서버에서 팀원의 출석, 휴식, 복귀, 퇴실 상태를 기록하고 하루 동안의 작업 시간을 확인할 수 있는 출석 관리 봇이다.

단순히 현재 출석 여부만 저장하는 것이 아니라, 사용자의 상태 변경 이력을 저장하여 작업 시간, 휴식 시간, 자리비움 시간을 계산할 수 있도록 설계한다.

---

## 2. 프로젝트 목표

### 1차 목표

- Discord 봇을 로컬에서 실행한다.
- 사용자가 `/출석` 명령어로 출석할 수 있다.
- 사용자가 `/퇴실` 명령어로 퇴실할 수 있다.
- 사용자가 `/내기록` 명령어로 오늘 작업 시간을 확인할 수 있다.
- 사용자가 `/상태` 명령어로 현재 출석 중인 팀원 상태를 확인할 수 있다.
- SQLite 파일에 활동 기록을 저장한다.

### 2차 목표

- 휴식 버튼을 추가한다.
- 복귀 버튼을 추가한다.
- Discord 음성 채널 퇴장 시 자동으로 자리비움 상태로 변경한다.
- Discord 음성 채널 재입장 시 자동으로 작업중 상태로 변경한다.
- Docker로 봇을 실행할 수 있게 한다.
- 무료 서버에 배포한다.
